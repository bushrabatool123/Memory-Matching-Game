Completed a basic web development assignment — a Memory Match Game using HTML, CSS & jQuery!
The game includes two difficulty levels, a move counter, timer, card flip animations, and sound effects. Cards are generated dynamically using jQuery, shuffled randomly, and matched through simple game logic — all split across three separate HTML, CSS, and JS files.
A fun little assignment that helped me practise jQuery selectors, DOM manipulation, and event handling hands-on. 🙌
